# Cangjie Language (倉頡編程語言) `cangjie/X.Y.Z`

WEMI supports Cangjie Installer/ZIP install. Supported Cangjie compiler infastructure with mingw64 based cjc and its llvm, and future Windows native cmake install MSVC build cjc/llvm compiler set.

- Cangjie 1.0.X
- Cangjie 1.X.Y

## \_\_future\_\_

Cangjie Lane as a open-sourced Language, with its compiler infastructure's documentation has defined cross-compile build and native build. WEMI will update contents with the binary installation with Cangjie update's its native Windows x64/arm64 build instructions.

- Cangjie Windows x64/ARM64 build, with cmake support
