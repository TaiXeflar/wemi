# MiHoYo/HoYoVerse SDK

Experimential MiHoYo SDK generating rules:

`mihoyo/hoyoverse` --> `mihoyo/<SDK>/<SERVER>`
`mihoyo/mihoyo` --> `mihoyo/<SDK>/<SERVER>`

Users must load main module `mihoyo/hoyoverse` or `mihoyo/mihoyo` to unlock specific SDKs.
Main module will include launcher.

## Honkai Impact 3 `miHoYo/BH3/<SERVER>`

MiHoYo Honkai Impact 3rd project modulefile is specified with out of game datacenter selection.
 - Global Specified severs are `jp`, `kr`, `sea`, `eu`, `us`, `tc`.
 - Mainland China server is `cn`.
