## Python 3

WEMI is not supported to generate Python 3 managment.

On Windows deployment, Python can have Installation achievment with different ways and dists via

 - PSF GUI Installer (to 3.15)
 - PSF CLI Installer (Experimential)
 - Anaconda, Miniconda, Conda-Forge, Intel Distribution for Python etc. Conda based GUI Installers
 - Visual Studio Installer (EOF)
 - Poetry
 - Pipenv
 - Astral UV    (Currently I recommends this)
 - PyPy
 - GraalPy
 - Rust-Python
