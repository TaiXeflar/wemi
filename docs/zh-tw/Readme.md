歡迎來到 環境模組安裝器 的中文文檔說明!

環境模組安裝器是一個基於Python語言、命令列的輕量級環境模組(Environment Modules)安裝程式。

## 注意事項
環境模組安裝器會調用調用`winreg`模組，查詢電腦的登錄編輯資料資料庫(Registry)。

如果你/你對此功能不信任，請勿使用環境模組安裝器的自動查詢功能，並且需要以自行設定的快取(`ModuleCache.txt`)執行環境模組的安裝。
